Hi All, Hi Stani,

I'm trying to make the speMac.blend (in fact the spe-editor script for 
blender) more a general solution for any flavor of Python and OS.

I'm running into the difficulty of determing the kind of install and os, 
and the code is getting longer and less elegant.

the solution I'm currently thinking of is to use the PYTHONPATH and/or 
PYTHON environment var, for the construction of the modules path and the 
location of wx.pth. Caveat is that this needs to be set.

the other option would be to use regular expression and scan through 
sys.path and get the site-packages path from there.

A combination of the two ways, i think, would be failsafe, so if no 
PYTHONPATH is found, do the regex over sys.path list.
