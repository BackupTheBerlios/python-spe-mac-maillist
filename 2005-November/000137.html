Another pre-announcement of SPE 'Kay release' 0.8.0.b

This release fixes the Windows installer and a lot of bugs, especially for
Mac.

:**Improved features**:

- find in files

:**Bug fixes**:

- Find dialog focus
- Workspace bugs (closing files)
- Installer (Windows)
- Sticky notes (Mac)
- Find in files tab (mac)
- Toolbar keyboard shortcut display (Mac)
- Kiki: regex console (Mac)

:**Installation**:

See http://pythonide.stani.be/manual/html/manual2.html There is now an
update section for MacOS X!

:**Contributors**:

- Thurston Stone (workspaces)
