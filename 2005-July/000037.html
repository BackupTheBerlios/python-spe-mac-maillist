Hello All,

The SVN repository for SPE is now on line. You can grab the latest code
using the following command.

svn checkout svn://svn.berlios.de/python/spe/trunk/

You can all so checkout the releases with this command:
svn checkout svn://svn.berlios.de/python/spe/branches/SPE-x.x.x

(Where the x's are the version)
Currently there aren't any releases yet and the trunk is 7.4.x.

Finally, you may browse and download parts of the source with the web
interface in the SVN section of this site.
