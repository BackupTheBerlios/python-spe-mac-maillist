This interface is not implemented yet, so I commented it out. Good you 
mention it. BTW this how I like to see error reports. You can still 
choose the old interface for mac 'single with tabs (mac, linux, 
windows)', but I wouldn't recommend it unless you are really tab 
addicted (tabs are un-mac if I understood well from mac users feedback). 
Now I'm developping the new one 'single with palette (mac default)', 
which is now the default for mac. Instead of a toolbar, there will be a 
tool palette (Photoshop like) to make it look more like a real, native 
mac app. This will be released in the 's' version. Of course I'm very 
curious if this works and I hope Mac users will be happy with that. I 
announce the release on this list.
