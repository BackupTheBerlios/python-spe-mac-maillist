Well, I will code the solutions for the list I
proposed. I would like SPE to be as comfortable as
possible for mac users. So it is not only about bugs,
but also about design and navigation...

Everytime I post a new release with enhancements for
the mac, I would like that a lot of you would test it
and report if things are fixed. So please test the
latest version (at least 0.7.4.o) and see if opening
files still gives a segmentation fault. If it does,
could someone put print statements to localize what
exactly causes this error.

Once SPE is acceptably stable, I also would like if
some of you put an effort in pushing it into the OS X
distributions.

What would be completely great if someone has
connections with Apple that they sponsor this project
with a mac with osX, so I could do the debugging
myself as well. Now it is hard to develop a mac app on
windows. Maybe a joint request would help.
